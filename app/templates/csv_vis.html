PQL - Plot Query Language

PQL is an SQL-like DSL I made for generating plots. The basic syntax is as follows:

PLOT <plot_type>
USING <x_column> [AS <x_label>], <y_column> [AS <y_label>]
[WHERE <condition>]
[GROUP BY <column>]

AST parser implementation can be found here: https://github.com/joeyshi12/pql-parser.
