This is a tool for creating plots from CSV data. Visualizations are generated using PQL - Plot Query Language: an SQL-like DSL I made for specifying plot configs.

The basic syntax is as follows:

PLOT <plot_type>
USING <x_column> [AS <x_label>], <y_column> [AS <y_label>]
[WHERE <condition>]
[GROUP BY <column>]
